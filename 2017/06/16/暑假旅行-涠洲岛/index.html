暑假旅行 -- 涠洲岛



我和瑶的暑假涠洲岛之旅。

这里展示的是我们的前期准备工作。

1. 总行程 + 必要费用统计

  • 6.27 出发
    • 胡东明：武昌10.41 – 南宁次日5.10
    • 肖瑶： 岳阳11.1 – 南宁次日5.10
      费用：共352.5
  • 6.28 到达涠洲岛
    • 南宁8.34 – 北海10.07
    • 做21路到客运港，坐12.30的船，14.00到涠洲岛
      费用：高铁60 + 船票来回600 + 登岛费120 = 780
  • 6.28 - 6.30 游玩
    住宿费：367（减免204）
  • 7.1 返程
    • 乘船10.10 – 11.20
    • 北海 - 南宁 - 岳阳次日5.38/武汉次日8.36
      费用：424元

综上：我们6.27上午出发，7.2日上午回到岳阳/武汉。总必要费用为1923.5元。

2. 岛上三天旅游安排（大致）

  • 岛上著名景点的高清彩图




  • 推荐路线/必去景点

    • 起早床去五彩滩看日出。
    • 滴水丹屏的海滩，有潜水、冲浪，水也比较干净。
    • 鳄鱼山公园。这个景点必须步行。可从山顶观光台眺望。

  • 小吃—某东的最爱

    • 还有很多很多……

3. 注意事项

  1. 岛上一般住宿的地方会有电瓶车出租，70左右/天。开车穿行整个岛大概40分钟。
  2. 可以自己去海鲜市场买东西，回来让房东帮忙做。
  3. 手机要买防水袋，要带防晒霜。
  4. 鞋子：拖鞋？
